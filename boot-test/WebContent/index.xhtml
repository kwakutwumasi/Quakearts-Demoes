Welcome to the Quakearts Demo Site. This site showcases some of the development libraries I have built over the years.

It also serves as an example of how quickly I can build apps in a short time with minimal code, using the "plumbing" code provided by my libraries and of course, the integration of other popular libraries/frameworks such as Hibernate, Weld, and Tomcat-Embedded.
